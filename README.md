## Getting Started

### Prequsites

1. Install rust
2. ```
   git clone https://github.com/chazfg/chatr.git
   cd chatr
   ```

### Running the server

```sh
cargo run --bin server
```

### Running the terminal client

```sh
cargo run --bin client
```

### Running the TUI client

```sh
cd client_chatr
cargo run
```
